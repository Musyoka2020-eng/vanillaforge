### 🔗 Connect Local Repository to GitHub

After creating the repository on GitHub, run these commands:

```bash
# Add remote origin
git remote add origin https://github.com/Musyoka2020-eng/vanillaforge.git

# Push to GitHub
git branch -M main
git push -u origin main
```
